class Monoid a => Group a where #

Group (S, *) is a monoid with inverse elements.

Requirements

  1. There's an identity element \(id\): \(a <> id = id <> a = a\).
  2. \(\diamond\) is associative: \((a <> b) <> c = a <> (b <> c)\).
  3. There's an inverse lement: \(a a^{-1} = a^{-1} a = 1\).
