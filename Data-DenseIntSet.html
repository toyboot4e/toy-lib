data DenseIntSet s #

Dense int set or a 64-ary tree that covers [0, n). = (Internal) segments

The idea is very similar to hibitset, with top and bottom reversed.

[.][.][.][.][.][.][.][.] ..  Layer 0: dense bits
[..........][..........] ..  Layer 1: sparse (1/64)
[......................] ..  Layer 2: more sparse (1/64^2)
