data Buffer s a #

A fixed-sized mutable vector with push/pop API.

Constructors

Buffer 

Fields

  • bufferVars :: !(MVector s Int)

    Stores the [front, back) position of the buffer in use. The front value is zero when the buffer is initialized as a stack or queue. The front value is at the middle of the internal buffer when initialized as a deque.

  • initialBufferPos :: !Int

    bufferVars initial values. Used in clearBuffer.

  • internalBuffer :: !(MVector s a)

    The storage.

  • internalBufferSize :: !Int

    The capacity of the buffer. It's doubled when initialized as a dequeue.
