Data.Graph.Dinic

Description

O(V^2E) max flow algorithm (Dinic's algorithm). Heavily inspired by cojna/iota.

Just run maxFlowD to get the result. All the other functions are used internally.

Overview

Dinic's algorithm loops the following steps: 1. runDinicBfs: Creates a DAG with BFS. 2. runDinicDfs: Adds flow from the source to the sink.
