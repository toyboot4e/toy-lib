newtype MUnionFind s #

Dense, mutable Union-Find tree (originally by `@pel`)

>>> stree <- newMUF 3
>>> sameMUF stree 0 2
False
>>> unifyMUF stree 0 2
True
>>> sameMUF stree 0 2
True
>>> unifyMUF stree 0 2
False
