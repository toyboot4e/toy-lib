Data.AdhocGraph

Description

Graph search with context for every vertex in the super graph.

Tree search doesn't need contect per vertex and thus not the target. But it would be great if we can use he same function for both trees and graphs.

Common cases

Search

  • [ ] DFS
  • [ ] BFS
  • [ ] Dijkstra

Search tree

  • [ ] Search paths

Search tree

  • [ ] DFS tree
  • [ ] BFS tree
  • [ ] Dijkstra tree
  • [ ] 01-BFS

Niche cases

  • [ ] Tree diameter
  • [ ] Walk through every path C - Remembering the Days
  • [ ] Come back to the first vertex
